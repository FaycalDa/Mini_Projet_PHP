# Mini Projet PHP – Back-office (Clients)

Mini application PHP/MySQL avec authentification et gestion de clients.

## Fonctionnalités
- Authentification (login/logout) via sessions
- Protection des pages (accès réservé)
- Clients : liste / ajout / modification

## Stack
- PHP 8
- MySQL (phpMyAdmin)
- PDO (requêtes préparées)

## Installation (local)
1. Créer une base MySQL `hotel_app`
2. Créer les tables `users` et `clients`
3. Configurer la connexion MySQL dans `src/database.php`
4. Démarrer Apache + MySQL (XAMPP)
5. Ouvrir `login.php` dans le navigateur
